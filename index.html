一键托管建议

  1. Netlify Drop（最省事）：把 zip 拖到 Netlify 的 “Deploy manually” 区域 → 自动给你一个 https 链接。
  2. GitHub Pages：新建 repo → 上传三个文件（index + 两个 html）→ Settings → Pages → 选择 main / root。
  3. Vercel / Cloudflare Pages：新建项目 → 选择静态站点 → 输出目录为仓库根目录即可。

注：页面使用了 Tailwind CDN；若要“离线可用/更稳”，需要把 Tailwind 打包到本地（我也可以帮你做离线版）。
